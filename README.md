## 🌟 Sobre o LuxeStay

O **LuxeStay** é uma plataforma completa para gerenciamento de reservas de hotéis. O projeto integra um frontend reativo com um backend robusto, permitindo que usuários naveguem por acomodações premium, verifiquem detalhes e realizem agendamentos de forma fluida.

### ✨ Funcionalidades
* **Exploração de Hotéis:** Cards detalhados com informações de preço, localização e avaliações.
* **Interface Responsiva:** Design adaptável para qualquer tamanho de tela (Mobile, Tablet e Desktop).
* **Fluxo de Reserva:** Experiência de usuário otimizada para conversão.
* **Segurança:** Implementação de autenticação e proteção de dados sensíveis.

---

## 🛠 Stack Tecnológica

### Frontend
- **React + TypeScript**: Desenvolvimento de componentes tipados e seguros.
- **Vite**: Ambiente de build ultra-rápido para melhor experiência de desenvolvimento.
- **Tailwind CSS**: Estilização utilitária para um design consistente e moderno.

### Backend
- **Node.js + Express**: Servidor escalável para processamento das requisições.
- **Autenticação JWT**: Segurança nas comunicações entre frontend e backend.
- **Arquitetura RESTful**: Organização de endpoints seguindo as melhores práticas.
